This page of the website is going to be all about tables, and how they work. Tables have a variety of functions, and as you can see, I have decided to make a calendar in order to showcase some of the capabilites of the table tag. The main component of the table tag of course is < table >, but there are also other tags that it uses, including < tr >, < td >, and < th >. < th > is the element used to define the heading of a table. < tr > is used to define a row in the table, and < td > is used to define a data value for the table. In the case of this calendar, the table heading, or < th > would be the days of the week. The < tr > would be each row in the calendar , and the < td > would be the individual squares in the table.



To make a table, we first specify the < table > tag. Then we list all of the table headings, or columns. After that, we use < tr > to specify each row in the table, and in the < tr > tag, we can include < td > tags to represent data. The picture below is a portion of my code for this calendar.
