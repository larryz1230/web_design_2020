Explanation

This is an example of an image map. An image map is basically a image that is separated into multiple sections. Below is a visual representation of one of these sections.

These sections are called areas, and they can be specified with the area tag < area >. The < area > tag has many attributes that can be specified including alt, coords, download, href, hreflang, media, rel, shape, target, and type. This outline of Africa does not look perfect, but this is because the outline is made up of connected coordinates. We can specify these coordinates by making an area tag, and adding a coords attribute. Then we list the coordinates, and the shape we want it to be. Some example choices of shapes are rectangles, circles, and poly which is just a bunch of connected dots.


An example area tag could look something like this: < area alt="" title="" coords= "1,2,5,32,3,12,2,43" shape="poly"< / area>. Inside of a given map element, there could be multiple areas that can be specified. The image below shows my code for the image map on this webpage. Of course I did not write all these coordinates, but instead used an image map coordinate marker website to retrieve the coordinates for each of the areas. This website is linked in the bibliography. To make the areas of the map clickable, I used some basic javascript to write text in the html document.
